# LPC Asset Attribution

All sprites in this directory are derived from the
[Universal LPC Spritesheet Character Generator](https://github.com/sanderfrenken/Universal-LPC-Spritesheet-Character-Generator)
project, which collects contributions from the
[Liberated Pixel Cup](https://opengameart.org/content/lpc-collection)
community on OpenGameArt.

Licensing varies per asset, generally CC-BY-SA 3.0+ and/or GPL 3.0+.
See the upstream `CREDITS.csv` for per-file artist credits.

This game's source code is independent; only the bundled PNG assets
under `public/assets/lpc/` carry the upstream license terms.
